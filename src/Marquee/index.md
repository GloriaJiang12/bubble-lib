# Marquee

## 介绍

通过使用 css 实现无缝滚动，包括有横向滚动和纵向滚动

## api

| 属性名称  |       说明       |  类型  | 默认值 |
| :-------: | :--------------: | :----: | :----: |
| direction | 滚动方向：x 和 y | string |  'x'   |
| duration  |     滚动时间     | number |   20   |
